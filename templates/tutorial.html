Running the Algorithm

The algorithms provided are:

  • Dijkstra's Algorithm
  • Bellman-Ford's Algorithm

Initiating the Algorithm

To get started, simply click the "Run Dijkstra" button or the "Run Bellman-Ford" button to run each respective algorithm.

After you clicked either button, depending on the algorithm you selected, either one or two pop-up boxes will appear asking for information. If you chose Djikstra's algorithm, then two pop-ups will appear, asking for the start and end node respectively:

Otherwise if you choose Bellman-Ford's algorithm, then it will only ask for the start node. Please enter an integer corresponding to the node's number.

Traversing Through the Algorithm

Once you have entered the input information correctly, The visualizer will enter into animation mode, which will visually display the chosen algorithm step by step. To control the stage of animation mode two control buttons will be made available.

You should now also see a table appear which details the current trace of the selected algorithm.

By pressing the "Step" button, it will proceed one step in the chosen algorithm, updating the table (highlighted in table) and also providing a visual representation in the graph.

Wrapping up the Algorithm

You can continue to press the "Step" button to go through the chosen algorithm until it finishes. By then, the graph will show the result of the algorithm and the table will be completed.

You can then press the "Exit Animation" button to leave animation mode and return back to edit mode.

Editing the Network

By default, there will be a pre-made network (undirected) made for you when you load the page. But, you can perform edit operations to modfiy the graph or create your very own network.

Clear

Press the "Clear" button to wipe the network

Graph State

There are two states for our network.
  • Unidrected Graph(default)
  • Directed Graph

The state is represented by a 'Star' icon in the the left of the buttons. The state currently being selected is highlighted in blue color.

To change the state to Directed, simply click on the "Directed Graph" button.Then it will be highlighted by blue, also "Undirected Grap" button will be unselected.

Add Node

To add a node to the network, simply click on the "Add Node" button. A node will be then generated with a number at its center.

You can also drag the nodes anywhere on the network editor.

Add Edge

To create an edge, click the "Add Edge" Button. You will be provided with a form where you can input the start node, end node and weight of the edge.

  • Provide the Start Node and End Node as integer number marked within the node in the network.
  • You can only enter negative weight for directed graph of Bellman-Ford Algorithm.
